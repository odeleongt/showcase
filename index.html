Prototype for MOOC records showcase (click to collapse or expand sections)

This page shows a prototype for showcasing your MOOC achievements in an attractive and simple interface. It is currently based on Coursera's achievements page styles and aims to concisely show the progress you have accomplished with your online education.

This is a work in progress, and if you are interested in sharing ideas or requesting features I would appreciate that you file an issue in the GitHub repository and browse those already submitted to contribute to what other interested people have already shared.

Once I have a few suggestions I'll start working on a more reproducible tool and publish the code on this repository, so you and others can use and improve it. In the mean time, you can watch the repository to get updates and know when showcase is ready for your use!
